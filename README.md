# Enterprise AI Platform

A full-stack enterprise AI platform built using FastAPI, React, LangChain, PostgreSQL, and modern AI agent architectures.

---

## 🚀 Features

### Core Chatbot
- AI-powered chatbot
- Conversational memory
- Streaming responses
- Multi-thread chat support

### Authentication
- JWT authentication
- Google OAuth login

### AI Features
- AI image generation
- PDF RAG chat system
- NL-to-SQL querying
- CSV / Excel / Google Sheets AI querying
- Research Digest Agent
- MCP Integration
- Tic Tac Toe AI Agent

### Data & Storage
- PostgreSQL
- ChromaDB vector storage
- File uploads
- Persistent chat history

```bash
cd frontend
npm install
npm run dev
```

---

## ▶️ Run Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```
